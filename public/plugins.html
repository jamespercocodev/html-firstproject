My Top 3 Audio Plugins

Fabfilter Pro Q 3

The Pro Q 3 is a very detailed eq. It is extremely powerful and allows for precise mixing and strong attention to detail. I would strongly recommend using this plugin for any newcomers.

Soothe 2

This plugin is a great plugin for unmasking audio

Fabfilter Pro R2

This plugin is a very great reverb
